# Park Fleur Special Assessment Simulator

This folder is ready for GitHub Pages.

## Quick publish steps
1. Create a **public** GitHub repository named either:
   - `yourusername.github.io` for a user site, or
   - any public repo name such as `park-fleur-assessment-simulator` for a project site.
2. Upload the files in this folder to the repository.
3. In the repository, go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**.
6. Save.
7. Wait for the site to publish.

## Resulting URL
- User site repo: `https://yourusername.github.io/`
- Project repo: `https://yourusername.github.io/repository-name/`
